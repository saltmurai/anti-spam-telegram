# TELEGRAM ANTI SPAM BASED ON TG-SPAM

### Instruction
- Step 1: Clone the repo
- Step 2: Install [TG-SPAM](https://tg-spam.umputun.dev/#installation)
- Step 3: Add a `.env` file in the folder with the following
```
TELEGRAM_TOKEN=...
TELEGRAM_GROUP=TELEGRAM_GROUP=...
```
- Step 4: Run `start.sh`
```
./start.sh
```
